
# Emotify_App

Requirements:
- python3.7
- pip install:
	* numpy
	* pandas
	* scikit-learn
	* spotipy
	* nltk
	* langdetect
	* lyricsgenius
	* fuzzywuzzy
	* python-Levenshtein

Also, need the following environment variables:
- GENIUS_API_KEY: token for querying Genius API
- SPOTIPY_CLIENT_ID: Spotify client ID (from developer account)
- SPOTIPY_CLIENT_SECRET: Spotify client secret
